Studi Kasus 2 - Library System

Run:
cd studikasus2
python3 main.py

Available features:

- Create author
- Create book
- Display book information
- Calculate late fee
- Create library member
- Borrow item
- Return item

Example:
Output setelah menjalankan perintah:
python3 main.py

Output:
=== Library System (Studi Kasus 2) ===

[Book] ISBN: ISBN12345, Title: Bumi, Author: Tere Liye
Author Age (2025): 46

Eka borrowed 'Bumi'
Late Fee (3 days): 4.5

Eka returned 'Bumi'
