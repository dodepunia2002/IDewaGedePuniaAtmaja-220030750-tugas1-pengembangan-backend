Studi Kasus 1 - University

Run:
cd studi_kasus_1
python3 main.py

Available features:

- Create address
- Create student
- List students
- Create professor
- Professor supervise student
- Assign address to student
- Purchase parking pass

Example:
Output setelah menjalankan perintah:
python3 main.py

Output:
=== University System (Without Framework) ===
